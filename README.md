# Chips List

Компонент для отображения списка чипов с адаптивным скрытием переполненных элементов. При нехватке места лишние чипы скрываются, а вместо них появляется кнопка `...`, по клику на которую открывается попап со всеми скрытыми чипами.

## Возможности

- **Адаптивность** - автоматический пересчёт видимых чипов при изменении размера контейнера (используется `ResizeObserver`)
- **Скрытие переполнения** - лишние чипы скрываются, а вместо них появляется кнопка `...` с попапом
- **Интерактивность** - поддержка клика по чипам с выделением выбранного (контролируемое состояние через `selectedId`)
- **Кастомизация** - настройка расстояния между чипами (`gap`), передача кастомных стилей и className
- **Модульность** - компонент отдельного чипа (`ChipButton`) можно использовать вне списка
- **Гибкость** - колбэк `onClick` получает выбранный элемент

## Установка и запуск

### Требования

- **Node.js** >= 18
- **npm** / **yarn** / **pnpm**

### Установка зависимостей

```bash
npm install
# или
yarn install
# или
pnpm install
```

### Режим разработки

npm run dev

### Сборка для продакшена

npm run build

### Сборка для предпросмотра

npx vite preview (Откроется по адресу http://localhost:4173)

### Быстрый старт для просмотра:

Запускаем команды в терминале из папки проекта поочередно

```bash
# 1. Установка зависимостей
npm install

# 2. Сборка проекта
npm run build

# 3. Запуск предпросмотра
npx vite preview

#4. Откройте ссылку в браузере
http://localhost:4173
```
